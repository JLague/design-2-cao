Voici les différentes étapes pour l'ouverture de l'outils C.A.O dans Matlab.

1- Lancez l'application Matlab version 2021b.
2- Ouvrir le fichier 'insert_name'. celui-ci devrait se trouver à l'endroit ou vous l'avez téléchargé.
3- Une fois le fichier ouvert dans Matlab, cliquer sur le bouton 'Run'
4- Inserer les paramètre désiré dans le simulateur.

Pour plus d'information concernant l'interface du simulateur, veuillez vous referer au Guide de l'utilisateur
dans le rapport.
